Toy CCG Parser
==============

OVERVIEW
--------

simple English CCG Parser


Python 2.x or 3.x
-----------------
toyccg is tested with

- CPython 2.7.5
- pypy3  2.4.0 (Python3.2.5 compatible)


USAGE
-----

`python toyccg/ccgparser.py < sentences.txt`
